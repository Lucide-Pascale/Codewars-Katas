<h2><a href=https://www.codewars.com/kata/65013fc50038a68939098dcf/train/javascript target="_blank">Party People</a></h2><h3>7 kyu</h3><p>Ava, Mark, Sheila, and Pete are at a party. However, Ava and Sheila are only staying if there are at least 4 people, Pete is only staying if there's at least 1 person, and Mark is only staying if there are at least 5 people. Therefore, Mark leaves, which makes Ava and Sheila leave, and Pete is left alone.</p><p>Given an array with the preferences of every person at a party for the minimum number of people present, determine how many people will stay.</p><h3 id="examples">Examples</h3><pre><code>[4, 5, 4, 1] ➞ 1# Ava's minimum number is 4, Mark's is 5, Sheila's is 4, and Pete's is 1.# Only 1 person (Pete) stays.[10, 12, 15, 15, 5] ➞ 0[2, 1, 2, 0] ➞ 4</code></pre><h3 id="notes">Notes</h3><ul><li>All attendees are included in the array.</li><li>Any person's count includes themself.</li><li>A persons preference minimum can be 0, 1, or any positive integer.</li><li>Expect valid input only.</li><li>0 &lt;= len(lst) &lt;= 10^5</li></ul>
